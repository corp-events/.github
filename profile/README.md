# corp-events

> Корпоративная платформа для управления мероприятиями

## О проекте

`corp-events` — это пространство для разработки продукта и связанных сервисов. Здесь собраны основные репозитории, инфраструктурные настройки и технические материалы.

## Репозитории

- `client` — фронтенд-приложение.
- `backend` — серверная часть и API.
- `infra` — инфраструктура и окружения (БД, Keycloak, мониторинг, CI/CD-конфигурация).

## Принципы

- Разделение по зонам ответственности: фронтенд, бэкенд, инфраструктура.
- Отдельные репозитории для независимых частей системы.
- Прозрачная структура для роста команды и проекта.

## Команды

- `frontend` — отвечает за клиентскую часть.
- `backend` — отвечает за API, бизнес-логику и хранение данных.
- `infra` — отвечает за окружение, деплой и системные настройки.

## Рабочий процесс

- Изменения вносятся через ветки и pull request.
- Обсуждение задач ведётся во внешнем трекере ([WEEEK](https://weeek.net/ru)).
- Доступ к репозиториям управляется через команды организации.
